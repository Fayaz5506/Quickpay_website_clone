Quickpay Link

Overview

Quickpay Link is a hosted payment page solution that allows you to accept payments with minimal integration effort. It's perfect for businesses that want to get started quickly or don't need a fully customized payment experience.

Features

  • Hosted payment page
  • Multiple payment methods
  • Mobile responsive design
  • Customizable branding
  • Automatic 3D Secure handling

Integration Steps

  1. Create a payment in your backend
  2. Generate a payment link
  3. Redirect the customer to the payment page
  4. Handle the callback when payment is completed

Example Integration

// Create a payment
POST /payments
{
    "currency": "DKK",
    "order_id": "order_123",
    "amount": 10000,
    "payment_methods": ["creditcard", "mobilepay"],
    "description": "Order #123"
}

// Response
{
    "id": "payment_id",
    "link": "https://payment.quickpay.net/payment/payment_id"
}
